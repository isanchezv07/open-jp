# Open-JP

**Open-jp** es una plataforma diseñada para el aprendizaje eficiente y estructurado del idioma japonés. Se centra en la organización detallada del contenido, la práctica activa y el uso de técnicas de estudio comprobadas.

## ¿Qué es Open-JP?
Es una aplicación web interactiva que ayuda a los estudiantes de japonés a estructurar su estudio mediante Unidades temáticas y Lecciones específicas, permitiendo un seguimiento claro del progreso. A diferencia de otras herramientas rígidas, Open-jp te da el control sobre cómo y cuánto estudias en cada sesión, adaptándose a tu propio ritmo.

## ⚠️ Nota Importante sobre el Reconocimiento Visual
> [!WARNING]
> **Imágenes de la API externa**
> Las imágenes utilizadas en los ejercicios de reconocimiento visual provienen de una API externa. Debido a esto, es posible que en ocasiones las imágenes tarden en cargar, no se muestren o fallen si la API experimenta intermitencias, límites de uso o cortes temporales. 

## Características Principales
* **Estructura Organizada:** El contenido está dividido en Unidades (ej. Hiragana, Kanji, Verbos) y cada unidad contiene Lecciones específicas.
* **Práctica Flexible y Multimodal:** Soporta diversos modos de práctica (lectura, escritura, reconocimiento visual, escucha) adaptables a cualquier lección.
* **Configuración Personalizada:** Puedes definir el número máximo de preguntas por sesión para adaptar la práctica a tu tiempo disponible.
* **Sistema SRS (Spaced Repetition System):** Utiliza algoritmos de repetición espaciada para asegurar la retención a largo plazo de los conceptos aprendidos.
* **Seguimiento de Progreso:** Analiza tus aciertos, errores y áreas de mejora a través de estadísticas al final de cada sesión.

## ¿Cómo funciona?
1. **Selección:** Desde el tablero principal, eliges la Unidad que deseas estudiar.
2. **Lección:** Dentro de la unidad, verás las lecciones disponibles.
3. **Configuración:** Puedes ajustar el número de preguntas que quieres resolver en esa sesión.
4. **Práctica Activa:** Eliges el modo (ej. lectura, escritura, reconocimiento visual) y comienzas la práctica. La aplicación rastrea tus aciertos y errores en tiempo real.
5. **Progreso:** Al finalizar, verás un resumen de tu desempeño indicando tu evolución.
